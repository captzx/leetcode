2. 利用该信息:The majority element is the element that appears more than ⌊ n/2 ⌋ times, 元素出现的次数 > floor(n/2)
   排序法，中间元素必然是众数  
   加减法，遇相同元素则+1，遇到不同元素则-1，如  
		元素： 2,2,1,1,1,2,2	 			元素： 2,2,2,2,1,3,4,5   
   		加减： 1,2,1,0,1,0,1				加减： 1,2,3,4,3,2,1,0	相对出现次数
   		众数： 2,_,_,1,_,2,_	 结果为2		众数： 2,_,_,_,_,_,_,_	结果为2
